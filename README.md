# StreetCleaning

second project for an introduction to AI

about multi-agent

Goals include:

1. render(by 5.18)
2. coordination between moving objects(by 5.21)
3. communication between agents(by 5.24)
4. maximize cleaning efficiency(by 5.29)
5. explore nn(by 6.5)

场景设想：
白天，热闹的街市，人和车和垃圾都挺多，传统方法，看到就扫，多了摇人。
晚上，街市空荡，只有垃圾，深度学习方法。
